Project Terbaru

Berikut adalah beberapa project yang saya buat dari mengikuti program SIB Batch 3 x Dicoding. Jika ingin melihat repostiori dan berkontribusi bisa klik saja di judul project.

Bookshelf App

Aplikasi ini untuk inventaris buku dengan fitur bisa menambah buku baru, menghapus buku, menandai sebagai selesai ataupun belum selesai dibaca, dan mencari buku berdasarkan judul.

Movies DB

Aplikasi ini untuk mencari semua film favoritmu, dibangun menggunakan APi dari OMDb, Fitur yang tersedia yaitu cari film, detail film.

Personal Note v.1

Aplikasi ini digunakan untuk membuat dan menyimpan catatan/note kamu. Memiliki fitur tambah,arsip,hapus catatan, batasan jumlah karakter pada judul, pencarian catatan berdasarkan judul.

Personal Note v.2

Aplikasi ini adalah pengembangan dari personal note sebelumnya dengan fitur tambahan yaitu dark mode, ubah bahasa, login register, dan semua fungsionalitas menggunakan APi.
